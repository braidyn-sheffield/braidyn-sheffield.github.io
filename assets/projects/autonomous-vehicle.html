Integration Robot

Home

The Integration Robot was developed to autonomously navigate a multi-challenge obstacle course using real-time sensor input and state-driven control logic. It combines ultrasonic distance sensors, TCS230 color sensors, a gyroscope, and servo/DC motors to perform tasks such as line following, object avoidance, and directional turning.

Built around the Teensy 4.1 microcontroller and programmed using Arduino C++, the system features modular code for motor control, sensor integration, and navigation states. Custom CAD parts were 3D printed, and a tailored power distribution system was designed using schematics and buck converters. This capstone project demonstrates comprehensive embedded systems integration, from mechanical and electrical design to software architecture and testing.

The robot features a custom chassis, sensor mounts, and servo brackets designed in Autodesk Inventor and fabricated using 3D printing. These drawings illustrate the mechanical design process and part integration.

  • Microcontroller: Teensy 4.1
  • Programming Language: Arduino C++
  • Development Environment: Arduino IDE
  • CAD Software: Autodesk Inventor
  • PCB/Schematic Design: KiCad
  • 3D Printing: Ender 3 and Prusa i3 MK3
  • Sensors: TCS230 Color Sensors, HC-SR04 Ultrasonic Sensors, MPU-6050 Gyroscope
  • Motor Control: XY-1600 H-Bridge Motor Driver
  • Power Management: Buck converters (12V to 7.2V and 5V)
  • Documentation: Microsoft Word, Google Docs
  • Version Control: GitHub
